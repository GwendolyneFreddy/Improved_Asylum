Components of the Mod

You may install any of these individually or together. They have all been tested with each other and all work together.

Component 1: Detectable Spells (by David W.)

This component is required in order for the others to function correctly.

By default, Baldur’s Gate II’s AI engine is remarkably simple. This component changes all of that. Detectable Spells does not include any game play per se, nor does it include anything that players are going to notice in the game. What it does do is enhances the AI capabilities of Baldur’s Gate II. This is necessary for the other components in the mod to function correctly. The other components in the mod as well as the harder fights and improved combat AI will not work if you do not install this component.

This is the same Detectable Spells enhancement that is found in many other combat-enhancing mods so if you have one of those installed then this may not be necessary. Please install it if you are not sure. There will not be any problems if you do.

Component 2: Improved Asylum

This is the primary component of the mod. It installs the new features of the mod including the improved fight scripts, new items, and other miscellaneous tweaks. Most notable among these are:

  • Resting in the Asylum is made more difficult by random difficult enemies given a 50% chance to appear on rest.
  • There is a party of adventurers in the previously empty room in the Asylum. You can get some basic information about Spellhold from them. You may also choose to fight them. WARNING: This is easily the most difficult fight in the game, and probably tougher than anything you've ever seen before. This fight is totally optional even if the mod is installed. If you do manage to win, though, they do have pretty awesome loot (although it has been significantly toned down from earlier versions of the mod).
  • The Umber Hulk prison is somewhat more challenging.
  • The face riddles are replaced, giving some new riddles.

Component 3: Spellhold Lich

This is exactly the same as the Spellhold Lich mod that I've been supporting for several years now. It adds a lich to the empty library in Spellhold, Level 1. If you have a previous version of Spellhold Lich, you will not be fighting against two liches at once. Installing this component and the previous one (Improved Asylum) will result in your having to fight against two liches at once. If you do not want that, then leave this component uninstalled. You will still have the fight against the lone lich that is included with Improved Asylum.

Component 4: Improved Caster Scripts for Players

Let's face it, the scripts that Bioware made for the game are designed for BG1, not BG2. None of them involve any high level spells, nor do they include intelligent casting.

Added here are one new script for mages and one for clerics. Both of these scripts concentrate on defense and healing first, then focus on offense. Both classes will use high level spells and they won't waste spells (No Horrid Wiltings on a lone goblin, for example).

How to Install

The Improved Asylum has an automated installation program. To install, simply run ImpAsylumV99.exe. The installation program will automatically find your BG2 installation and extract all of the necessary files to the directories where they need to be. It will then fire up the standard WeiDU program which you would use to install the mod the same as you would any other WeiDU mod. You will be able to select which of the three components that you wish to install. You may install one, two, or all of them.

If you do not install all components, but decide to install the additional ones at a later time, simply run the setup-impasylum program and choose to install them this time.

Uninstallation Instructions

Run the setup-impasylum executable found in your main BG2 directory (the same directory that contains dialog.tlk). It will give you the option to uninstall. Note that this will not affect games already in session. I.E., if you've been to the Asylum, uninstalling won't do anything to your current game.

Compatibility Concerns

This mod is compatible with all known WeiDU mods, except for those that overwrite the Asylum. There should not be any issues if Improved Asylum is installed last. I have tested it with, and verified compatibility with the following mods:

- Kelsey
- Solaufein
- Oversight
- The Unusual Oddities Shop
- RPG Dungeon Item Pack
- Kitanya
- Unfinished Business (if UB is installed second)
- Banter Packs
- Flirt Packs
- Tactics Improved Mage Scripts
- Sword Coast Stratagems II (if SCS2 is installed first)

If you find a mod that does conflict with Improved Asylum, send an e-mail to:  rastor@rpgdungeon.net with "Improved Asylum" in the subject line and I'll either fix the problem or include a warning with the mod.
